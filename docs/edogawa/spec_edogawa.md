## 0. このドキュメントについて

丸亀ツールの手法を江戸川に展開した検証記録です。江戸川は**全国唯一の河川コース**で、水面特性が丸亀（瀬戸内のプール型）と根本的に違います。丸亀で確定した信号定義（①〜⑬）はそのまま使いつつ、**江戸川固有の検証（潮・風・まくり・ST・初日）**で「効く／効かない」を実データで切り分けました。

結論を先に：**ももやんの穴党勘（STのばらつき）が本物の初日エッジを掘り当て、仕組みから攻めた潮・まくり分析は空振り**という、ワークフローの価値を示す回になりました。

## 1. データ土台（2026-07構築・再現可能）

- **Kファイル**：丸亀と同じ `https://www1.mbrace.or.jp/od2/K/{YYYYMM}/k{YYMMDD}.lzh`（全国一括）。365日分を5〜8並列DL（存在＋サイズ非ゼロでスキップ）。
- **江戸川ブロック**：`03KBGN` 〜 `03KEND` で切り出し。**184開催日**（残り181日は非開催）。
- **パース済みレース数**：2,043R（総エントリー12,422÷6）。1日平均11.3R。
  - 「コース数不揃い」35日は**欠場・出走取消**（江戸川は全国一欠場が多い）。総エントリー÷6のレース数計算は正しく機能。
- **Bファイル（締切時刻）**：`.../od2/B/{YYYYMM}/b{YYMMDD}.lzh`、`03BBGN`〜`03BEND`。締切時刻は `電話投票締切予定HH：MM`（全角→NFKC正規化）。**全2,208R取得成功**。
- **江戸川はデイタイム開催（締切10〜16時）**。丸亀のオールナイトと違い、潮位サイクルのサンプリングが昼帯に偏る（後述の潮検証で重要）。

### 潮位データ（東京検潮所 TK）

- URL：`https://www.data.jma.go.jp/gmd/kaiyou/data/db/tide/suisan/txt/{年}/TK.txt`（天文潮位・年間ファイル・約50KB）。
- フォーマット：固定長。24時×3桁(1-72) + 年2桁(73-74) + 月2桁(75-76) + 日2桁(77-78) + 地点(79-80)。
- **天文潮位を採用**（実測でなく）。理由＝天候由来のうねりを含まず、潮の駆動力そのものを表す＝風との二重計上を避けられる。
- **流速の代理変数 d潮位/dt（cm/h）**を各レースに付与（中心差分）。江戸川で効くのは水位でなく「流れの速さ」なので、上げ/下げ・高低だけでなく変化速度を主軸に据えた。

### イン残差の基準（顔ぶれ補正）

- 顔ぶれ期待＝1コース選手の**全国1コース逃げ率**（全24場Kファイルから自前計算、1コ20走以上、不足は全体0.552で代替）。
- **江戸川イン残差の基準線：実測イン1着率46.7% − 顔ぶれ期待51.5% ＝ −4.8pt**。
  - この−4.8ptが**江戸川の水面ペナルティ**（顔ぶれで説明できないイン不利）。丸亀（イン55.5%）と別世界。以降の検証はこの残差の**変動**が潮・風・日目で動くかを見る。

## 3. 見送り／棄却シグナル（根拠つき）

### 3-1. 潮（流れ）＝見送り

- 仮説：河川コースなら潮位の上げ下げが実際の水流になり、丸亀（プール型で空振り）と違って効くはず。
- 検証：全2,075Rに締切時刻の潮位・流速(d潮位/dt)を付与。上げ/下げ×流速×風向×風速で総当たり＋回帰。
- 結果：
  - 潮単独・流速単独はすべて1σ未満。
  - 最初に見えた「下げ潮×速い +6.4pt(1.9σ)」は、そのセルの**約83%が北系風（インを浮かせる風）**の交絡だった。
  - **風を固定すると潮の差はゼロ**（南系風の中で上げvs下げ＝0.0σ、北系風でも1.4σ）。
  - 回帰：風だけでR²=0.0075、潮を足しても0.0085（増分+0.001）。潮関連の項は全て|t|<2。
- **結論**：潮は単独でも風との組合せでも、インの沈みに独立効果なし。丸亀の潮検証（仕様書5-2）と同じ構図。最も手間のかかる外部パイプラインを根拠を持って作らずに済んだ。
- 補足：流れの変動幅（毎時±35cm程度）がスタート・ターンを実質的に動かすほど大きくない、または動かしても腕・整備で吸収される。

### 3-2. まくり（向かい風×3・4コース）＝棄却

- 仮説：南の向かい風でまくり艇（3・4コース）が壁を壊し、高配当が出るのでは。
- 検証：決まり手（結果ヘッダー行末尾）＋1着艇コースを2,074Rでパース。江戸川まくり系決着30.9%（丸亀26.8%より高い）。
- 結果：
  - **南風はまくりを増やさない**：南系31.1%(+0.2σ)・北系31.3%・基準30.9%。強風6m+はむしろ28.8%(−0.9σ)。
  - 3・4コース勝率も南風で上がらない（3コース南14.6% vs 北15.2%）。
  - **南風でインが沈んだぶんの受け皿は2コース**：南系−北系で1コース−4.3pt(1.9σ)、**2コース+3.5pt(2.0σ)**、3〜6は全て1σ未満。
- 機構：向かい風は1コースの逃げを削ると同時に3・4のまくり足も殺す（第1マークまで伸びない）→隣で構える2コースが差すだけで得。**江戸川の南風は「まくりの風」でなく「2コース差しの風」**。
- **結論**：まくり仮説は棄却。受け皿の2コースは最も人気で配当が安く穴妙味に乏しい。
- 丸亀⑫との接続：南風の江戸川＝水面全体が「2コースが食う型」に傾く条件。買うなら2アタマ（2-1・2-3流し）、⑫「自分が食う型」選手が2枠の南風レースが濃い。

### 3-3. 風（南系イン不利）＝効くが織り込み濃厚（本命サイド情報）

- 南系−4.3pt(2.3σ)・北+5.8pt(2.2σ)・東+11.2pt(2.3σ)・強風6m+ −3.2pt。回帰で南系t=−2.6で有意。
- ただし「江戸川は南の向かい風でイン受難」は常連全員の常識＝織り込み済みの筆頭。残差が実在してもオッズに入っていればEVゼロ。
- 用途：穴の起点というより、南風＝イン信頼度を下げる／東風＝鉄板(本命サイド)の**事前フィルタ**。過小評価かどうかはオッズ待ち（§5）。

## 4. 確定シグナル：江戸川固有（ももやん発案・ST機構）

### ★ 機構の全体像

```
STがばらつく（江戸川=全国1位）  ← 根本原因（ももやんの直感）
   ├─ 初日は特にバラバラ（SD0.088・遅れ33%）
   │     └→ インが確実に逃げられない → 初日イン崩壊（残差−10.6pt）★タイミング穴
   └─ 水面全体でインが噛み合わない
         └→ 6コースが残る（全国最高・3連率+4.7pt）＝水面定数（土台フォーム）
```

### E-1. STばらつき（全国1位）＝根本原因

- スロー勢（コース1-3）のST標準偏差を全24場比較。
- **江戸川 ST-SD=0.0796＝堂々の1位**。2位常滑0.0722を大きく引き離す明確な外れ値（2位〜11位の差より1位〜2位の差が大きい）。丸亀は11位0.0685。
- 「スタートがばらつくのが江戸川」はデータが太鼓判。以下の初日崩壊・6コース残りの両方がここから生える。

### E-2. 節内ST収束（初日バラバラ→終盤締まる）

| 日目 | 平均ST | ST-SD | 遅れ(.20超)率 |
|---|---|---|---|
| 初日 | 0.174 | 0.088 | 33.2% |
| 2日目 | 0.172 | 0.081 | 30.0% |
| 3日目 | 0.161 | 0.080 | 26.5% |
| 4日目 | 0.161 | 0.075 | 26.6% |
| 5日目 | 0.155 | 0.072 | 22.8% |

- 平均STもSDも遅れ率も**単調に締まる**。丸亀⑨「初日は荒れる」の機構そのもので、江戸川は水面が読みにくいぶん初日効果が強い。

### E-3. 6コース連帯率（全国最高）＝水面定数・土台フォーム

| | 2連率 | 3連率 |
|---|---|---|
| 江戸川 | 11.0% | 27.1% |
| 丸亀 | 9.6% | 25.5% |
| 全国 | 8.4% | 22.4% |

- 江戸川は丸亀（既に外残り水面）をさらに上回り、6コース3連率は全国比**+4.7pt**。
- **重要**：6コース残りは**日目では動かない**（初日24.6%・3日目30.1%・6日目31.1%とバラバラ、初日+2日目26.0%≒4日目以降26.4%＝0.2σ）。＝「初日だから跳ねる」信号でなく、**江戸川で常に6をヒモに置く土台フォーム**。過小評価かはオッズ待ち。

### ★ E-4. 初日signal（今回の最重要・EV検証最優先）

| 日目 | n | 実イン1着 | 顔ぶれ期待 | 残差 | 基準差 |
|---|---|---|---|---|---|
| **初日** | 370 | **40.5%** | 51.1% | **−0.106** | **−5.8pt(2.3σ)** |
| 2日目 | 376 | 44.9% | 50.8% | −0.058 | −1.1pt |
| 3日目 | 378 | 48.9% | 51.9% | −0.030 | +1.8pt |

- 初日は、座った選手の全国実力が51.1%の逃げを期待させるのに**実際は40.5%しか勝てない＝顔ぶれで説明できない−10.6ptのイン不利**。
- 初日−終盤(4日目〜)の残差差＝**−0.077(2.6σ)** で有意。**丸亀⑨初日(−5pt)の2倍の深さ**。
- 用途：**江戸川初日＝1号艇を強く嫌う日**。1号艇を頭から消す／相手を広げる。ヒモに6コース（E-3の土台フォーム）を厚く。
- 注意：「初日は荒れる」は常識でもある＝織り込み済みか過小評価かはオッズ次第（§5のEV検証最優先項目）。

**三条件ルールの判定**

**因果経路**

- 江戸川は後半戦にA級を集める番組編成（顔ぶれ期待が9R以降 56→65%と急上昇）。前半戦はB級中心。ここまでは全場共通。
- **江戸川固有＝顔ぶれ補正後もなお前半が沈む**。6Rは期待50.4%に対し実測34.9%＝**−15.5ptで全番号中最大の乖離**。
- 解釈：B級中心の前半戦は**ST精度が最も低い層**が集まる時間帯。E-1（江戸川のST分散は全国1位）がそのバラつきを増幅し、実力（全国逃げ率）が同じでも江戸川ではB級のインが余計に守れない → 顔ぶれ補正で消えない残差が残る。**E-1のST機構から一貫して説明できる**。
- 6Rは企画レースではない（レース名構成は5R・7Rと同一＝予選120/一般55）。企画レース由来の説明は否定済み。

**重ね掛け（初日と独立に積み上がる）**

| 初日全体 | 370 | 40.5% | −5.9pt | −2.37 |
| **2-6R × 初日** | 160 | **32.5%** | **−9.4pt** | **−2.59** |

初日(−5.9) と 前半ゾーン(−3.3) がほぼ加算される（≒−9.4）＝**独立した2軸**。
**江戸川で最も濃い穴条件＝「初日の前半戦（特に6R）」**。
なお 6R は 2日目以降でも −8.9pt(z=−2.32) と単独で生きており、初日依存ではない。

## 5. オッズ答え合わせリスト（EV検証待ち）

**オッズ蓄積：6場一括で稼働中**（`data/odds/03/YYYY-MM-DD.csv` に毎朝、3連単確定オッズ全120通り）。

- ※パース注意：高配当セルは `class="oddsPoint is-fColor1"` 等の追加クラスが付くため `class="oddsPoint[^"]*"` で拾うこと（`oddsPoint\s*"` だと万舟目を取りこぼし、120通り揃わずレース丸ごと欠落する選択バイアス）。任意の1ファイルで「1R=120行そろっているか」を検品。

1年蓄積後、同じ前半学習・後半テストのウォークフォワードで以下をEV（確率×オッズ）検証：

1. **【最優先】初日の1号艇は過小に嫌われているか**（E-4）。実力40.5%に対し初日1号艇のオッズが安すぎ＝過剰人気なら、初日の1号艇切り（頭≠1）が+EV。丸亀⑨は識別力実証済み・江戸川初日はその2倍深いので最有望。
1-b. **【最優先タイ】6R・前半戦ゾーンは織り込まれているか**（E-5）。6Rのイン実測34.9%に対し1号艇オッズがどう値付けされているか。**「初日 × 2-6R」（実測32.5%）が最濃条件**なので、この重ね掛けセルを最初に測る。
2. **6コース残りは適正価格か**（E-3）。丸亀11-5では高率6の3着側が割高（流しマネー理論）。江戸川の全国最高6残りが織り込み済みか、初日×6ヒモが過小か。
3. **南風×2コース差しは過小評価か**（3-2）。⑫「食う型」選手が2枠の南風レースで2アタマ（2-1・2-3流し）が+EVか。配当が安いので天井は低い前提。
4. **南風のイン割引が過剰か不足か**（3-3）。

## 7. 総括

- 江戸川の2大「仕組み仮説」（潮・まくり）はどちらも**根拠を持って切れた**（潮＝風の交絡／まくり＝受け皿は2コース差し）。負けシグナルを2本確定できたのは、ノイズを足さずに済む収穫。
- 一方、**ももやんの穴党勘（STのばらつき）が系統分析の見落とした本物の初日エッジを掘り当てた**。①STばらつく②節内で締まる③6コース残る、の3直感は全部本物で、STのばらつきを根本原因に互いに因果で繋がっていた。
- **江戸川は穴エッジが薄い水面ではなかった**。初日signal（残差−10.6pt・丸亀⑨の2倍）という、EV検証最優先の候補を得た。
- さらに**レース番号構造（E-5）も採用**され、当地エッジは「初日1本」から**2軸**になった。しかも両者は独立に積み上がり（初日−5.9＋前半ゾーン−3.3 ≒ 重ね掛け−9.4）、**「初日の前半戦（特に6R）」が江戸川最濃の穴条件**として特定できた。
- 江戸川の全発見が**ST機構ひとつに収束**した点が特筆される：ST分散全国1位（E-1）→ ①初日は特にバラつく（E-2→E-4）②水面全体で6が残る（E-3）③B級中心の前半戦でインが守れない（E-5）。単一の根本原因から3系統のシグナルが導出され、いずれも顔ぶれ補正後に生き残った。
- 次：オッズ1年蓄積→§5の答え合わせ。運用は当面「**初日 × 2-6R（特に6R）＝1号艇を強く嫌う**／6コースを土台ヒモ／南風・東風で本命サイド判断は人間が直前」。
