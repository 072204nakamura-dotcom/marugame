## ■ この書類の使い方（ももやんさん向け・PC操作）

プログラミングの知識は要りません。次の順でそのまま進めてください。

1. **PCでターミナル（黒い画面）を開く**
   - Mac：Launchpad →「ターミナル」
   - Windows：スタートメニュー →「PowerShell」
2. **リポジトリのフォルダに移動する**（`marugame` を置いてある場所へ）
   ```bash
   cd ~/marugame
   ```
   ※置き場所が違う場合は、フォルダをターミナルにドラッグすればパスが入ります
3. **最新の状態にする**
   ```bash
   git pull
   ```
4. **Claude Code を起動する**
   ```bash
   claude
   ```
5. **下の「■ Claude Code に渡す指示文」を全部コピーして貼り付け、Enter**
6. 一緒に **3つのファイルを同じフォルダに置いておく**（Claude Code が読みます）
   - `戸田_穴党ツール_仕様書.md`
   - `戸田_4コースまくり屋表.csv`
   - `戸田_6コース残し表.csv`

> 途中でClaude Codeが「実行していいですか？」と聞いてきたら、内容を見て `y`（はい）で進めてください。
> **`git push` の直前だけは必ず一度止めて**、変更内容を確認してから許可してください。

### 2. 作るもの（既存会場と同じ流儀で）

添付の `戸田_穴党ツール_仕様書.md` が唯一の仕様源です。これに沿って以下を作ってください。

#### (A) 日次ビルドスクリプト（戸田版）

既存会場のビルドスクリプトと同じ場所・同じ命名規則で作成。中身は仕様書の**セクション5**の通り、以下2つの選手テーブルを**毎朝再生成**します。

- **戸田_4コースまくり屋表**：全国365日分のKファイルから、選手×コース別に［走数・1着・2着・3着・ST合計・ST件数・まくり系1着］を集計 → 地力（全コース通算3連対率・30走以上）→ 地力ビン（0.05刻み）別の「4コースまくり系1着率」期待カーブ → 各選手の実まくり率をベイズ収縮（K=15、期待カーブへ寄せる）→ **まくり力 = 収縮値 − 期待値（pt表記）**。全国4コース15走未満は除外。
- **戸田_6コース残し表**：同様に地力ビン別「6コース2-3連対率」期待カーブ → ベイズ収縮（K=15）→ **残し残差 = 収縮値 − 期待値（pt表記）**。全国6コース15走未満は除外。

添付CSV（`戸田_4コースまくり屋表.csv` / `戸田_6コース残し表.csv`）は**2026-07-19時点のスナップショット**です。列構成の見本として使い、**再生成した結果がこれと近い値になるか**を必ず突き合わせてください（完全一致はしません。日付窓がずれるため）。

重要：**決まり手（逃げ/まくり/差し/まくり差し/抜き/恵まれ）の取得が必須**です。仕様書セクション7-1の通り、各レースの列見出し行（`着 艇 登番…` を含む行）の**末尾**にあります。`line.split('ﾚｰｽﾀｲﾑ')[-1]` から全角スペースを除去して取得してください。

#### (B) 当日出走表スクリプト（戸田版）

既存会場と同じ方式で、当日のBファイルから戸田の出走表を取得し、上の2表を登録番号（regno）でJOINして、レースごとに次を出力するJSONを作ってください。

**判定ロジック（仕様書セクション6と完全に一致させること）**

```
【朝の判定・出走表だけで確定】
4号艇（枠なり前提）の選手 → まくり屋表を引く
  まくり力 >= +8   → フラグ「まくり屋カド」 スコア +2
  まくり力 <= -4   → フラグ「カド消し」     スコア -1
6号艇の選手 → 残し表を引く
  残し残差 >= +6   → フラグ「6残す」        スコア +1
  残し残差 <= -6   → フラグ「6切り」        スコア  0（ヒモ整理の表示のみ）

【買い目方針テキストの生成】
・「まくり屋カド」あり かつ「6残す」あり
    → 本線: 4-6-x / 4-5-x / 4-1-x
       注記: 「二枚重ねは過去1年でn=11のためサンプル不足。単独シグナル2本の重なりとして扱う」
・「まくり屋カド」のみ
    → 本線: 4-5-x / 4-1-x / 4-3-x（4コースまくり時の2着は5コース26%・1コース・3コース）
       注記: 「1号艇はまくり決着なら着外57%。頭は4、1は薄く」
・「6残す」のみ
    → ヒモ強調: 6を2-3着に厚く。まくり決着なら2着づけ、まくり差し決着なら3着づけ
・「カド消し」
    → 注記: 「4コース勝率6.7%。カドを消してヒモを絞れる」
・フラグなし
    → 「見送り」
```

**表示に必ず載せる当地ベース値（仕様書セクション1・4より）**
- 戸田のイン1着率 42.9%（全国55.2%）／万舟率 17.6%／3連単中央値 3,320円
- まくり系決着 38.3%（まくり26.5%＋まくり差し11.8%）
- 6号艇：2着なら中央7,900円・万舟40.0%／3着なら中央4,540円・万舟19.6%

既存会場のページのデザイン・構造をそのまま踏襲してください。載せる内容：
